# TimeUtil
Simple, free, open source utility to calculate dates made on Unity using C Sharp.
Try it now! https://hecthor4.github.io/TimeUtil/

# About this
There's currently two versions in this repository, the HTML one and the Unity one.
I have tried to recreate them as close as possible to my originate idea, which only was an ejercise to test my programming abilities.
It is obvious that the HTML versions is more polished and better designed compared with the Unity version.
